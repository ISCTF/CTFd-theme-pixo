# CTFd-theme-pixo

Origin CTFd core-theme theme: [core-theme](https://github.com/CTFd/core-theme)

Origin CTFd-theme-pixo theme: [CTFd-theme-pixo](https://github.com/hmrserver/CTFd-theme-pixo)

Rewritten CTFd-theme-pixo based on core-theme

## Install

Install Git and execute command in CTFd root directory

```shell
cd /path/to/CTFd
git clone https://github.com/ISCTF/CTFd-theme-pixo CTFd/themes/pixo
```

## Develop

1. Install Git and yarn.
2. Clone this repo.
3. Run `yarn insatll` to install packages.
4. Run `yarn build` to build.
